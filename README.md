# ProfileEditor

Customize the Minecraft account's skin, name, and cape directly through ZenithProxy.

*NOTE: These changes apply to your minecraft.net profile. Be careful before making any changes, especially if other people have access to the command input (owner role) on your proxy.*

<img width="255" height="277" alt="Screenshot_1" src="https://github.com/user-attachments/assets/51719300-f4bb-4eb3-9ccd-835d082bd8e5" />

## Commands
(.profileEditor | .pe)
* `pe status | .pe s`
* `pe name <newName>`
* `pe skin <classic/slim> [file/URL]`
* `pe cape [number/off]`

## Installation

Drop the plugin jar into your ZenithProxy `plugins` folder and restart, or run this command: `plugins download https://github.com/tr330ne/ProfileEditor/releases/download/19.07.26/ProfileEditor-19.07.26.jar`
